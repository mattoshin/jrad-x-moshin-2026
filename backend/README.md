<h2>Running This application </h2>
<h1>Pre requiste:</h1>
<ol>
<li>PHP</li>
<li>Composer</li>
<li>Node</li>
</ol>

<h1>Upon cloning this application, please CD into ocean/backend and run the following commands</h1>
<ol>
<li>NPM I (Node Package Manager Install)</li>
<li>Composer require </li>
<li>Composer i </li>
</ol>
